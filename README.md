# HR-Data-Analytics-
## Employee day-to-day data insights visualization 
---
This project visulize the sales data of a employees over past 3 months for the year 2022 considering following aspects
- Total Presence %
- Work from home (WFH) %
- Sick leave (SL) %
- Employee wise presence, WFH and SL percentage
- Employee wise day-to-day data
- Presence % trend over everyday
- WFH % trend over everyday
- SL % trend over everyday
> Insights:
- In which day of the week total employee presence % is high
- In which day of the week, eployees take WFH more often
- In which day of the week, employees take the more SL
